# Estonian Accounting Standards (RTJ) & Compliance Reference

## Company Size Categories

A company qualifies for a category if it meets **2 of 3** thresholds on the balance sheet date:

| Category | Balance Sheet Total | Revenue | Employees | Required Components |
|---|---|---|---|---|
| **Micro** | <= EUR 175K | <= EUR 50K | <= 10 | Abridged BS + IS, up to 3 notes |
| **Small** | <= EUR 4M | <= EUR 8M | <= 50 | BS, IS, up to 9 notes, management report |
| **Medium** | <= EUR 20M | <= EUR 40M | <= 250 | All 4 statements + full notes (~15) + management report |
| **Large** | Above medium | Above medium | > 250 | All 4 statements + full notes + management report (may require IFRS) |

**Micro additional criteria**: Liabilities must not exceed equity, and must have only one shareholder who is also the board member.

## Required Statements by Size

| Statement | Micro | Small | Medium | Large |
|---|---|---|---|---|
| Balance Sheet (Bilanss) | Abridged | Full | Full | Full |
| Income Statement (Kasumiaruanne) | Abridged | Full | Full | Full |
| Cash Flow Statement (Rahavoogude aruanne) | No | No | Yes | Yes |
| Changes in Equity (Omakapitali muutuste aruanne) | No | No | Yes | Yes |
| Notes (Lisad) | Up to 3 | Up to 9 | ~15 | ~15+ |
| Management Report (Tegevusaruanne) | No | Yes | Yes | Yes |
| Profit Allocation Proposal | Yes | Yes | Yes | Yes |

## Audit / Review Requirements (from 2025)

| Requirement | Thresholds (2 of 3 for 2 consecutive years) |
|---|---|
| **Statutory audit** required | Revenue > EUR 5M, assets > EUR 2.5M, employees > 50 |
| **Statutory audit** also if any single | Revenue > EUR 15M, assets > EUR 7.5M, employees > 150 |
| **Review** required | Revenue > EUR 1.6M, assets > EUR 0.8M, employees > 24 |
| **No audit/review** | Below review thresholds |

## Validation Checklist

### Financial Statement Integrity
- [ ] Trial balance balances (total debits = total credits)
- [ ] Balance sheet balances (Total Assets = Total Liabilities + Total Equity)
- [ ] Net profit in IS matches current year profit in equity section
- [ ] Cash flow reconciles (net change = closing cash - opening cash)
- [ ] Equity changes: opening + changes = closing for each component

### Year-End Completeness
- [ ] All 12 fiscal periods closed
- [ ] Depreciation run complete for full year
- [ ] All accruals posted (wages, taxes, expenses)
- [ ] Bank reconciliations complete
- [ ] Intercompany balances confirmed (if applicable)

### Comparative Figures
- [ ] Prior year figures match last filed annual report
- [ ] Any restatements properly disclosed in notes

### Compliance
- [ ] Correct company size category applied
- [ ] All required statements included for size category
- [ ] Required number of notes complete
- [ ] Management report included (if not micro)
- [ ] Profit allocation proposal included
- [ ] Related party transactions disclosed
- [ ] Revenue breakdown by EMTAK code included

### Filing
- [ ] Board member signatures obtained
- [ ] Audit/review report obtained (if required)
- [ ] Filed before June 30 deadline
- [ ] Confirmation received from Business Register

## Common Issues

| Issue | Resolution |
|---|---|
| Trial balance doesn't balance | Check for posting errors, run GL reconciliation |
| Prior year comparatives don't match | Verify prior year was not restated; check for audit adjustments |
| Statutory reserve below minimum | Include transfer in profit allocation proposal |
| Missing related party disclosures | Query shareholder/board member transactions |
| Balance sheet doesn't balance | Check retained earnings, verify current year P&L flows |
| Cash flow doesn't reconcile | Compare cash GL detail to BS cash lines; check FX |
| Incorrect size category | Reassess 2-of-3 thresholds |
