# Set Up Entity for Year-End Bookkeeping

Guide me through setting up the accounting structure for my company:

1. Ask which legal entity and fiscal year
2. Ask if I have a chart of accounts from a prior accountant or accounting software (like Merit Aktiva)
3. If yes: ask me to paste or upload it, then import it as-is
4. If no: set up the standard Estonian IFRS template and ask about my business type to customize
5. From the chart of accounts, identify bank-related accounts
6. Ask me which banks my company uses (LHV, SEB, Swedbank, Wise, etc.) and get IBAN for each
7. Create bank accounts in the system, linked to the correct GL accounts
8. Configure posting profiles for the transaction types my company needs
9. Show a summary of everything that was set up
