# e-Business Register Filing Guide

## Overview

| Attribute | Detail |
|---|---|
| **Portal** | ariregister.rik.ee |
| **Authentication** | ID-card, Smart-ID, or Mobile-ID |
| **Format** | XBRL (primary) or portal form entry |
| **Deadline** | 6 months after fiscal year end (June 30 for Dec 31 year-end) |
| **Penalties** | Warning, then potential forced dissolution for repeated violations |

## Submission Methods

### Method 1: XBRL File Upload (Recommended)

Upload pre-generated XBRL files directly to the portal. This is the fastest and most reliable method — the files contain all financial data in machine-readable format, eliminating manual data entry errors.

**Required files** (generated by `/artifi-ee:generate-xbrl-file`):
- `Vormid_{REGCODE}.xsd` — companion schema declaring which forms/notes are included
- `Aruanne_{REGCODE}.xbrl` — instance document with all financial data

**Upload steps:**
1. Log in to ariregister.rik.ee with ID-card or Smart-ID
2. Navigate to "Aruanded" (Reports) section
3. Create new report — select fiscal year period
4. Choose **"Lae XBRL fail"** (Upload XBRL file) option
5. Upload both files (the `.xsd` companion schema and the `.xbrl` instance document)
6. The portal validates the files against the et-gaap taxonomy — fix any validation errors
7. Review the generated report preview — verify all numbers are correct
8. Add text sections if needed (management report narrative, accounting policies text)
9. Add signatories — board members who will sign
10. Digital signature — each board member signs with ID-card/Smart-ID (PIN2)
11. Audit step (if required) — assign auditor, auditor signs in portal
12. Submit to Business Register

**Validation errors**: If the portal reports errors during upload, common causes include:
- Missing required elements (check company size requirements)
- Balance sheet doesn't balance (Assets != Liabilities + Equity)
- Missing contexts referenced by facts
- Element names not matching taxonomy (regenerate files with corrected mapping)

### Method 2: Portal Form Entry (Manual Fallback)

Log in, create report, fill structured forms. The portal auto-tags data with XBRL taxonomy. Use the mapping table from **xbrl-mapping.md** to know which value goes into which field.

**Steps:**
1. Log in to ariregister.rik.ee with ID-card or Smart-ID
2. Navigate to "Aruanded" (Reports) section
3. Create new report — select fiscal year period
4. Select company size category (micro/small/medium/large)
5. Choose report forms based on size category
6. Fill in financial data:
   - Title page (company details, board members)
   - Balance sheet (all line items)
   - Income statement scheme 1 or 2
   - Cash flow statement (if required)
   - Statement of changes in equity (if required)
   - Notes
   - Management report text
   - Profit allocation proposal
   - Revenue breakdown by EMTAK activity code
7. Validate — portal checks mandatory fields and cross-references
8. Generate PDF preview — review before signing
9. Add signatories — board members who will sign
10. Digital signature — each board member signs with ID-card/Smart-ID (PIN2)
11. Audit step (if required) — assign auditor, auditor signs in portal
12. Submit to Business Register
13. Download confirmation and signed copy

## After Submission

- Report becomes public record in the Business Register
- Confirmation email sent to company
- Changes/corrections possible via resubmission (within certain timeframes)
- Statistics Estonia (Statistikaamet) receives data automatically

## Preparation Timeline (Dec 31 Year-End)

| Phase | Timing | Activities |
|---|---|---|
| Year-end close | January | Complete month-end close for December, final adjustments |
| Draft financials | February | Generate trial balance, draft balance sheet and P&L |
| Notes preparation | Feb-Mar | Compile all note disclosures |
| Management report | March | Draft narrative sections, calculate KPIs |
| Audit/review | Mar-Apr | External auditor fieldwork (if required) |
| Board approval | Apr-May | Board reviews and approves the report |
| XBRL generation | May | Generate XBRL files using `/artifi-ee:generate-xbrl-file` |
| Shareholder meeting | May-Jun | General meeting approves annual report |
| Filing | By Jun 30 | Upload XBRL files to Business Register via portal |
