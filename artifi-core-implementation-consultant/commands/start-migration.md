# Start ERP Migration

Act as my implementation consultant and guide me through migrating to Arfiti:

1. Ask which legal entity we're migrating (help me find it if I don't know the ID)
2. Ask if I'm starting from scratch or migrating from an existing accounting system
3. If migrating: ask which system (Merit Aktiva, QuickBooks Online, SmartAccounts, or other)
4. Ask for the cutoff date — the date the old system stops and Arfiti takes over
5. Check if I already have a chart of accounts set up in Arfiti
6. Ask about my business type to understand what data I'll need
7. Calculate if there's a gap period between cutoff and today
8. Present a migration plan with the recommended path and phases
