# Check Migration Status

Show me the current status of my ERP migration:

1. Ask which legal entity we're checking
2. Recall the migration state from agent memory
3. For each phase, check actual progress:
   - **Phase 0 — Discovery**: Was a migration path chosen?
   - **Phase 1 — Foundation**: Are accounts, posting profiles, and bank accounts set up?
   - **Phase 2 — Master Data**: How many vendors, customers, items, employees imported?
   - **Phase 3 — Opening Balances**: Is the trial balance posted and balanced? Are subledger records created?
   - **Phase 4 — Gap Period**: Are transactions posted for the gap period (if applicable)?
   - **Phase 5 — Verification**: Do all reports balance?
4. Present a clear dashboard with completion percentages
5. Recommend the next action

Use the Implementation Consultant skill for reference on what each phase requires.
