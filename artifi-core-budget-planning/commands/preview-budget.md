# Preview Budget

Show me what a budget would look like without creating anything:

1. Ask which legal entity, fiscal year, and scenario
2. Ask the method (scratch with growth, copy-forward, forecast from actuals)
3. Fetch source data (prior year actuals, existing budget, employee records)
4. Calculate budget amounts using the chosen method
5. Show the preview table by account and period
6. Show total revenue vs total expense and net margin
7. If employee budget: show per-employee cost breakdown with burden rates
8. If project budget: show per-project profitability summary

Do NOT create any budget version or submit any lines. This is preview-only.

Use the Budget Planning skill for the full workflow.
