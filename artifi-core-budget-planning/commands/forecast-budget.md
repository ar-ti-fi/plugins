# Forecast Budget

Help me create a forecast revision:

1. Ask which legal entity and fiscal year
2. Find the approved/locked budget for that year
3. Fetch YTD actuals from GL
4. Show me YTD actual vs budget comparison by account
5. Ask which forecast method to use (run rate, trend, seasonal, manual, budget-to-actual blend)
6. Calculate projected amounts for remaining months
7. Show me full-year forecast table: actuals (past months) + projections (future months)
8. Show comparison to original budget: total variance and key account differences
9. After I confirm, create the forecast version and submit lines
10. Ask if I want to submit for approval

Use the Budget Planning skill for the full workflow.
