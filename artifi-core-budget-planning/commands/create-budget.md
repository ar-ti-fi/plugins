# Create Budget

Guide me through creating a budget step by step:

1. Ask which legal entity and fiscal year
2. Ask the scenario type (budget, forecast, stretch)
3. Ask the granularity (monthly, quarterly, annual)
4. Ask whether to build from scratch, copy forward from prior year, or use a template
5. For scratch: ask about growth assumptions and seasonal patterns
6. For copy-forward: ask what adjustments to make (global %, per-account overrides)
7. Fetch prior year data and calculate amounts
8. Show me a preview table of all budget lines by account and period
9. Show summary: total revenue, total expenses, net income
10. After I confirm, create the budget version and submit lines
11. Ask if I want to add dimension tags (department, project, employee)
12. Ask if I want to submit for approval now
